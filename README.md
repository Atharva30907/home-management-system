# 🏠 Home Management System

> ✨ A simple, elegant, and powerful task management app built using Python

---

## 🚀 Overview

The **Home Management System** is a lightweight productivity tool designed to help users manage daily household tasks efficiently.

Built with **pure Python using Streamlit**, this project removes the complexity of traditional web development and delivers a clean, interactive UI directly from Python.

---

## 🌟 Features

* 📝 Add daily tasks instantly
* ❌ Delete tasks with one click
* ⚡ Real-time updates
* 🎨 Clean and modern UI
* 🧠 Beginner-friendly architecture
* 🌐 Runs directly in browser

---

## 📦 Installation

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/home-management-system.git

# Navigate to project folder
cd home-management-system

# Install dependencies
pip install streamlit
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```

Then open in browser:
👉 http://localhost:8501
